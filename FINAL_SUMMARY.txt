## 🎓 Understanding the API

Once running, you get these endpoints:

```
GET  /health                          → Check if server is running
GET  /api/resources/supported         → List supported resources
POST /api/transform                   → Transform CSV to FHIR
POST /api/validate                    → Validate CSV data
GET  /docs                            → Interactive API documentation
```

## 🆘 If You Get Stuck

1. **Installation fails?** 
   → Try: `pip install -e ".[dev]"` instead of uv

2. **Port 8000 in use?**
   → Try: `python -m uvicorn src.csv2fhir.server:app --port 8001`

3. **Tests fail?**
   → Check: Dependencies installed with `uv sync`

4. **Need help?**
   → Read: `GETTING_STARTED.md` → "Troubleshooting" section
